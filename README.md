# 🔥 FocusForge

FocusForge is a minimal productivity web app designed to help you:
- Set a daily goal
- Track Pomodoro sessions
- Build consistent focus habits

## 🚀 Features
- 25-minute Pomodoro timer
- Session counter
- Clean dark UI
- No dependencies

## 📦 How to Use
1. Clone the repository
2. Open `index.html` in your browser
3. Start focusing

## 🛠 Future Improvements
- LocalStorage persistence
- Custom timer lengths
- Progress charts
- Weekly analytics
- Mobile responsiveness
